Install Theme

To install the theme:

  1. Please log in to your Shopify store, click Online Store.
  2. At Theme Library, click Add theme, choose Upload zip file.
  3. The Upload Theme dialog displays. Click Add File or drop file to upload to select the .zip file you want to upload.
  4. Click Upload file.
  5. Wait for the uploading process finishes, you can select Publish button to publish what you have just uploaded to be an active theme.

Updating theme without losing old settings

To know what has been updated for our theme, please go to this link

Remember: Our instruction below is only applied for updating theme source without losing settings in Theme Editor. All of customizations you made in js or code files will not be updated and you need to re-do. Apps you installed into your site will not be updated and you need to reinstall.
Ella has been updated to support Section on every page, we changed structure of pages, some functions such as Dynamic Filter, Pages. When updating theme, it will only help you to keep the settings of Homepage and some other parts, you may need to set up for other pages again.

If your store is using from Ella 3.0.0 version and you would like to update to newest theme source to have all of new features or new layouts. Please follow below instruction:

Firstly, please backup your current theme source. You should duplicate your current theme

Then, go to Edit Code -> open this file settings_data.json and COPY ALL

After that, please go to the newest theme source which you just installed into your site. Go to Edit Code -> open this file settings_data.json , delete and PASTE code you just copied into this file

If you copy settings in setting_data.json file in your old theme source, please search for "enable_multilang" in file and change "true" to "false" to be able to use the new Shopify Languages function
